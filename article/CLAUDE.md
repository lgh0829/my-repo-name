# 이 공간에 대하여

개인 블로그(티스토리) 발행을 위한 글감 제조 공간입니다.
아이디어 발굴부터 발행 직전 원고까지, 하나의 주제가 하나의 파일로 관리됩니다.

---

# 독자 및 글의 성격

- **주요 독자:** 기획자, 일반인
- **글의 성격:** 실무 노하우 / 기술 분석 / 에세이 — 파일마다 다를 수 있음
- **언어:** 한국어
- **발행 채널:** 티스토리

---

# 글감 생애주기

| 상태 | 설명 |
|---|---|
| `아이디어` | 주제와 핵심 메시지만 있는 상태 |
| `윤곽` | 목차 또는 구성이 잡힌 상태 |
| `초안` | 본문 작성 중 |
| `검토` | 완성된 초안, 퇴고 중 |
| `발행대기` | 발행 직전 완성본 |
| `발행완료` | 티스토리에 올라간 글 |

---

# 파일 규칙

## 파일명
- 영문 소문자 + 하이픈
- 주제를 간결하게 반영
- 예: `ux-writing-basics.md` / `ai-pm-collaboration.md`

## Frontmatter 필수 항목
```
aliases: 한글 제목
작성일: YYYY-MM-DD
수정일: YYYY-MM-DD
상태: 아이디어 | 윤곽 | 초안 | 검토 | 발행대기 | 발행완료
유형: 실무노하우 | 기술분석 | 에세이
태그: 소문자 영문 (쉼표 구분)
발행일: YYYY-MM-DD (발행 전은 빈값)
```

---

# Claude에게 요청할 수 있는 것

- **아이디어 단계:** 주제 구체화, 핵심 메시지 정리, 독자 관점 점검
- **윤곽 단계:** 목차 구성 제안, 흐름 검토
- **초안 단계:** 본문 작성 보조, 문단 다듬기, 예시 추가
- **검토 단계:** 퇴고, 어색한 문장 수정, 제목 후보 제안
- **완성 단계:** 티스토리 발행용 마크다운 정리, 요약문 작성

# 스타일 가이드

- 어조는 글의 유형에 따라 조정 (에세이는 부드럽게, 기술 분석은 명료하게)
- 기본 문체: 경어체 (습니다/합니다)
- 전문 용어는 처음 등장 시 괄호로 부연 설명
- 소제목은 독자가 스캔할 수 있도록 내용 요약형으로 작성
