## 검색 실행 흐름 요약

```
검색 입력
├── 상세 검색식 입력
│   └── 문법 오류 감지 → 수정 제안(사용자 승인)
│       └── Elastic Search → 2.3 검색 결과
└── 자연어 입력
    └── AI 충분도 판단
        ├── 충분 → Vector DB (Top 50) → 2.3 검색 결과          [경로 A]
        └── 불충분
            └── 클러스터링 → 상위 토픽 Q&A 폼
                ├── 50건 이하 → Vector DB (Top 50) → 2.3 검색 결과   [경로 B-1]
                └── 50건 초과 → 필터/검색식 보강 → 2.3 검색 결과      [경로 B-2]

2.3 검색 결과
├── 채팅 (새 주제 / 이어서 질문)
├── 검색 결과 조회 (필터링 / 정렬 / 상세 조회)
└── 분석 대상 문서 담기 (추가 / 제외 / 분석 리포트 / 상세 조회)
```

## Reference — Claude.ai 채팅창 IA

> PatSol 채팅창 설계 레퍼런스. 동일 패턴은 그대로 적용, 차이점은 PatSol 적용 방향 참고.

### IA 구조

```
Claude.ai
├── 좌측 사이드바 (고정)
│   ├── 새 대화 버튼
│   ├── 프로젝트 목록 (폴더형 그룹핑)
│   └── 대화 히스토리 목록 (날짜 그룹: 오늘 / 어제 / 지난 7일…)
│
└── 메인 채팅 영역
    ├── 상단 바 (고정)
    │   ├── 모델 선택 드롭다운
    │   └── 공유 버튼
    │
    ├── 대화 스레드 (스크롤)
    │   ├── 사용자 메시지 (우측 말풍선)
    │   └── AI 응답 (좌측, 마크다운 렌더링)
    │       └── 아티팩트 패널 (코드·문서·SVG 등 우측 분리 표시)
    │
    └── 프롬프트 입력창 (하단 고정)
        ├── 텍스트 입력 필드 (멀티라인, 자동 높이 확장)
        ├── 파일 첨부 버튼
        ├── 도구 토글 (웹 검색 등)
        └── 전송 버튼
```

### PatSol 적용 방향

| Claude.ai 요소 | 설명 | PatSol 적용 방향 |
|---|---|---|
| 사이드바 히스토리 | 날짜별 그룹핑, 세션 제목 자동 생성 | 동일 패턴. 세션 제목은 AI가 20~40자로 자동 생성 |
| 모델 선택 | 상단 고정 드롭다운 | 단일 모델 → 불필요. 새로 조사하기 / 히스토리 컨트롤로 대체 |
| 아티팩트 패널 | 코드·문서를 채팅 우측에 분리 표시 | **매직노트 탭**과 동일 역할. 분석 리포트·청구항 초안 분리 표시 |
| 멀티라인 입력 | 자동 높이 확장, Shift+Enter 줄바꿈 | 동일 적용. 특허 청구항 입력 특성상 필수 |
| 대화 스레드 | 위→아래 선형 흐름 | 동일. 단 경로 B 의도 구체화 시 칩 선택 UI 추가 |
| 파일 첨부 | 채팅창 내 멀티모달 입력 | 현재 스코프 외. 문서 Import는 검색 시작 화면에서 분리 처리 |
