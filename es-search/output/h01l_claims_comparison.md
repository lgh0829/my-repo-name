**같은 H01L 반도체 분야임에도 스타일 차이가 명확히 존재.** 연결어·구분자는 기술분야와 완전히 무관한 사무소 관행 차이. 독립항 전략과 기재 밀도는 사무소 전략이 주요인이며 기술분야가 일부 영향.

## ⚠️ 리포트 분석과의 불일치 발견

이 샘플은 두 리포트의 핵심 분석 결과와 **정반대**로 나타남.

| | 리포트 분석 결과 | 샘플 실측 |
|---|---|---|
| jang | 장치 우선 (장치:방법 = 1.54배) | 장치+방법 쌍 |
| leeandmok | 장치+방법 쌍이 표준 (각 985·999회) | 장치+장치 (방법 없음) |

**원인 추정**: 리포트의 말미 빈도 통계는 전체 청구항(독립항+종속항)의 말미 단어를 집계한 것. 이 방식으로는 실제 독립항 전략을 정확히 파악하기 어려움.

- jang의 방법 청구항 상당수가 이 샘플처럼 장치를 인용하는 **카테고리 교차 독립항** 형태일 가능성 → 말미는 "방법"으로 잡히지만 실질은 장치 종속적
- leeandmok의 일부 출원은 이 샘플처럼 **같은 카테고리 복수 독립항** 전략 사용 → 방법 말미 카운트가 낮아질 수 있음

> **후속 검토 필요**: 독립항만 따로 분리해 말미 패턴을 재집계해야 리포트 수치의 정확성 검증 가능.
