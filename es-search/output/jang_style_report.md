### 1-5. Embodiment 전개 방식

- **실시형태/실시예 언급**: 1,000건 샘플의 서브샘플 100건 전부(100%)에서 "실시형태" 또는 "실시 형태" 발견, 평균 **47.3회**. 단, 전체 데이터(53,925건) 기준으로는 **실시예가 우세** (일본법인 실시예 8,468건 > 실시 형태 4,924건)
- **JP 번역 출원 전용 도입 정형구 3종** ※ 아래는 일본 원문 직역 패턴. 순수 국내 출원 프롬프트에 그대로 사용 금지:

```
① "이하, 본 발명의 실시 형태에 대해 도면을 참조하여 설명한다."   ← JP 번역체
② "본 발명의 실시형태에 대해서 도면을 사용하여 자세히 설명한다."  ← JP 번역체
③ "이하, 본 발명에 대하여, 도면을 적절히 참조하면서 상세하게 설명한다." ← JP 번역체
```

- **전개 순서 (정형화)**: 도면 목록 소개 → 구성 설명(도면 번호 기반) → 동작 원리 → 변형 실시예
- '실시형태'·'실시 형태'는 일본 특허법 "実施の形態" 직역 표현. JP 번역 출원에서 집중 등장하며, 신규 순수 국내 출원에서는 '실시예'로 대체할 것 [데이터 확인]

### 1-6. 문체 특성 (장수길 고유 특성)

> [데이터 확인] 전체 출원의 59.0%(31,811건)가 일본 법인 출원인(가부시키가이샤/가부시끼가이샤)이며, 나머지 비일본법인 중에도 도쿄엘렉트론·미쯔비시 등 일본 기업이 다수 포함되어 JP→KR 번역 출원이 스타일의 실질적 기반임이 확인됨. 아래 패턴은 번역 출원에서 비롯된 것으로, 신규 순수 국내 출원 프롬프트에 그대로 적용하면 번역체로 인식될 수 있음.

**일본어 번역체 식별 패턴**

| 패턴 | 예시 표현 | 대응 일본어 |

### 3-1. id:3 | Claims Generation (우선순위: 높음)

**수정 후**
```

[청구항 작성 규칙]
```
