AI 출력 규약 — Do / Don't

✓ DO
  • [거절이유] / [주의] / [참고] 태그로 검토 등급 명시
  • 법령은 § 접두사로 시작
  • 청구항 번호는 #N 형식
  • ## h2로 섹션 구분
  • 수정 제안은 수정 제안: 접두사 사용
  • 액션 버튼은 [반영] 대괄호 형식
✕ DON'T
  • # h1 사용 — 위계 혼란
  • HTML 태그 직접 삽입
  • 연속 빈 줄 2개 이상
  • 이미지 마크다운 ![]()
  • 법적 확정 표현 ("등록 가능합니다")
  • 수정 내용 동의 없이 자동 적용
  • 1. ol로 청구항 번호 나열 — #N 칩 사용
  • `인라인 코드`로 청구항 번호 참조 — #N 칩 사용

function MessageBlockRenderer({ block }) {
  switch (block.type) {
    case "text":         return <MarkdownBlock content={block.content} />;
    case "verif_result": return <VerifCard {...block} />;
    case "action_prompt":return <ActionPrompt {...block} onAction={handleAction} />;
    case "legal_cite":   return <LegalCiteBlock {...block} />;
  }
}
