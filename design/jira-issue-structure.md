## 의존 관계

```
C (페르소나/유저저니) ──► D (우선순위 결정) ──► 개발 Story
A (기능명세서) ──────────►┘
B (와이어프레임) ─────────►┘
```

- **C → D**: 페르소나·플로우 확정 후 워크플로우 우선순위 선정 가능
- **A + B → D**: 기능명세서·와이어프레임 완료 후 구현 난이도 추정 가능
- **A ↔ B**: 병렬 진행 가능
