## 해당 서비스가 필요한 기업

- 특허 출원 전 경쟁 특허가 있는지 직접 파악하기 어려운 기업

- 경쟁사 기술 동향을 파악하고 기술 공백을 발굴해야 하는 기업
- 특허 출원을 변리사에게 의뢰하기 전, 발명서 작성이 필요한 기업

## 주요 기능

### 1. 자연어·자료 기반 특허 검색
- 문장, 논문, 발명 자료를 그대로 입력하여 유사 특허 자동 탐색

### 2. AI 검색 결과 분류
- 검색된 특허를 발명 목적(Object)과 해결 수단(Solution)에 따라 기술 카테고리별로 자동 분류

### 3. 문헌 Q&A 챗봇
- 특허 문서에 직접 질문을 입력하면 핵심 내용을 즉시 요약·답변
- 긴 특허 명세서를 처음부터 읽지 않아도 필요한 정보를 확인 가능

### 4. 특허 동향 분석 및 시각화
- **특허기술흐름도** — 연도별 기술 진화 흐름 한눈에 파악
- **특허 동향 분석** — 기술·국가별 출원 트렌드
- **OS 매트릭스** — 기술 공백 시각화 (기회 영역 발굴)
- **중요 특허 요약** — 핵심 청구항·도면 자동 요약
- **활용 특허 요약** — 라이선싱·회피 설계 참고용
- 분석 결과는 Word·PDF로 즉시 다운로드해 내부 보고서로 활용 가능

## 활용 방안

| 직무 | 활용 시나리오 |
|------|-------------|
| **IP R&D 담당자** | 경쟁사 기술 동향 파악, 기술 공백 발굴 및 전략 수립 |
| **연구자** | 선행기술 탐색, 발명 아이디어 참고 자료 수집 |
| **경영진** | R&D 투자 방향 설정, 사업화 가능성 검토 |

**주요 사용 시나리오**

- **신제품 개발 전** — 관련 특허를 빠르게 검색해 침해 리스크와 기술 공백을 동시에 파악
- **경쟁사 모니터링** — 경쟁사 출원 동향을 정기 분석해 기술 전략 방향 수립
- **특허 출원 검토** — 기존 특허와의 차별성을 분석해 출원 가능성 사전 검토
- **기술 이전·라이선스** — 관련 특허 범위 및 활용 가능 특허 파악

## 서비스 강점

1. **청구항 기반 특허 검색** — 특허의 기술 권리를 결정하는 청구항을 중점으로 검색하고, 유사도를 분석해 관련 선행특허를 놓치지 않음
2. **국내·미국 특허 통합 검색** — 한국어 입력으로 한국/미국 특허를 함께 검색 가능. 일본·중국·유럽 순차 추가 예정
3. **통합 워크플로우** — 검색 결과를 바탕으로 전략 보고서를 생성하거나 청구항 설계 전략 분석을 한 플랫폼에서 실시할 수 있으며, Word·PDF 다운로드로 사내 검토 자료로 바로 활용 가능
4. **검색식 없이 즉시 사용** — 문장·논문·발명 자료를 그대로 입력해도 유사 특허를 검색하므로, 특허 전담 인력 없이도 즉시 활용 가능
5. **검색 결과 자동 분류 및 순위화** — 검색 결과가 관련도 순으로 정렬되어 중요한 특허를 빠르게 찾고 조사를 마칠 수 있음. AI 자동 분류로 선행기술조사 누락 리스크를 낮출 수 있음
6. **데이터 보안** — 고객 데이터는 외부 AI 학습에 활용되지 않으며, 국내 AWS 인프라에서 격리 운영됨 (AWS 보안 인증 적용)
