# V1(기본) vs V2(검색의도 반영) 추출 키워드 비교

**사용자 검색 의도:**
> 전기차 주요 장치의 데이터 스캔 장치로부터 데이터를 획득하고, 데이터 DB를 통해 선별 압축 데이터를 획득하고, 선별 압축 데이터를 클라우드로 전송하여, 클라우드는 OBD 기반 데이터를 획득하고, 선별 압축 데이터 및 OBD 기반 데이터를 포함하는 전기차 관련 데이터에 기초하여 이슈 데이터를 추출하고, 추출된 이슈 데이터를 통해 이슈 이벤트를 설정하여 실시간 알림 및 경고를 제공하는 기술

### 특허번호: KR1020180037969_청구항_0
**[V1(기본 추출)]**
- 목적: 유지보수 이슈 식별, 사용자 보고 통합, 운영 신뢰성 향상
- 수단: 센서 기반 성능검출, 모바일 이슈 수신, 유지보수 이슈 전송

**[V2(검색 의도 반영)]**
- 목적: 유지보수 이슈, 실시간 알림, 원격 정비
- 수단: 센서 성능데이터, 사용자 이슈보고, 이슈 판단
