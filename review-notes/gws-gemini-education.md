## 1. 한 줄 요약

> Google Workspace의 주요 서비스에 Gemini AI를 연동하여 반복 업무를 자동화하고, 이미지·영상 생성 및 노코드 앱 빌더까지 활용할 수 있다.

---

## 2. 교육 내용

### 1) 내용 요약

**Google Workspace와 Gemini 연동**
- **Gmail** — 메일 히스토리 요약, 빠른 답장 제안
- **Drive** — 파일 검색, 파일 내부 자료 요약
- **Docs / Slide / Sheet** — 문서 작성 보조
- **Gem** — 특정 역할의 AI 어시스턴트를 직접 만들어 활용 가능

**Gemini 주요 툴 3종**
| 툴 | 용도 |
|---|---|
| Deep Research | 광범위한 탐색 및 원천 자료 수집 |
| NotebookLM | 자료 내재화 및 맞춤형 지식창고 구축 + 시각화 |
| Deep Think | 심층 논리 분석 및 최종 결과물 기획 |

**Workspace Studio**
- 반복 작업 자동화 워크플로우 세팅 가능
- 키워드 기반으로 워크플로우 실행 가능

---

**AI Ultra Access 고급 기능**

- **Nano Banana (Gemini 2.5 Flash Image)** — 멀티모달 이미지 생성 (무료 하루 100장), 프롬프트 4원칙: 주제 / 동작 / 구도 / 배경
- **Google Flow** — 이미지·영상의 원하는 부분만 선택하여 수정, 시작·끝 프레임 기반 영상 생성
- **Google Whisk** — 업로드 이미지 + 프롬프트로 애니메이션 영상 생성, 피사체/장면/스타일 유형 선택 가능
- **Opal** — 반복 업무를 단계별로 실행하는 노코드 AI 빌더, 아이디어 기반으로 실제 동작 앱 생성 가능 (단, 데이터 보안·사용량 제한 주의)
- **Antigravity** — 계획 → 작업 → 실행 → 브라우저 테스트까지 자동 수행, 병렬 에이전트 실행 지원

---

### 2) 인상적인 점 / 실무 적용 포인트

- **도면 이미지 부분 수정** — Nano Banana로 생성한 도면 이미지에서 사용자가 수정을 원하는 영역만 선택하여, 해당 부분을 도면 설명에 맞게 변경하는 방식으로 활용 가능
- **Gem으로 가상 변리사 구성** — Gem을 활용해 변리사 역할의 AI를 만들고, PatSol 서비스 기능에 대한 임시 평가를 받아보는 용도로 활용 가능
- **Gemini 무료 기능 활성화** — Gmail, Drive, Docs 단위로 Gemini를 무료 범위 내에서 활성화하여 팀 업무에 적용 검토 필요

---

## 3. 우리 팀에 적용할 수 있는 것

| 적용 아이디어 | 관련 업무 | 우선순위 |
|---|---|---|
| Gemini 무료 기능 활성화 (Gmail, Drive, Docs) | 팀 전체 업무 | 높음 |
| Gem으로 가상 변리사 구성 | PatSol 서비스 기능 검증 | 중간 |
| Nano Banana 도면 이미지 부분 수정 | 도면 생성 기능 기획 | 낮음 |
