# 애니파이브 × Wand (PatSol) — 1차 기술 제안서

---

## 1. 제안 범위

| 항목 | 내용 |
|------|------|
| 직무발명서 생성 기능 | 멀티모달 입력 → AI 초안 생성 → 편집 → 다운로드 |
| 특허 VectorDB 구축 | 애니파이브 보유 특허 DB 전체 임베딩 + PatSol 원격 운영 |
| 연동 방식 | REST API |
| 인프라 소유 | 애니파이브 클라우드 계정 (내재화) |
| 운영 주체 | PatSol 원격 운영·유지보수 |

---

## 2. 요구기능리스트

### 2-1. 직무발명서 생성

| # | 기능 | 비고 |
|---|------|------|
| F-01 | 멀티파일 업로드 | 이미지·PDF·PPT·DOCX, 동시 다중 업로드 |
| F-02 | 이미지 파싱·해석 | 멀티모달 LLM 적용 |
| F-03 | 문서 텍스트 추출 | PDF·PPT·DOCX 파서 |
| F-04 | 직무발명서 초안 생성 | 표준 양식 기반 LLM 프롬프팅 |
| F-05 | 편집 가능한 미리보기 | 리치 텍스트 에디터 |
| F-06 | 다운로드 | DOCX·PDF·HWP (HWP 필수) |
| F-07 | REST API 제공 | generate / edit / export 엔드포인트 |

### 2-2. 특허 VectorDB 구축

| # | 기능 | 비고 |
|---|------|------|
| V-01 | 특허 데이터 수집 파이프라인 | 애니파이브 IP 커버리지 전체 |
| V-02 | 특허 문서 임베딩 | 청구항 중점 + 공보 항목별 청크 분할 |
| V-03 | VectorDB 구축·배포 | 애니파이브 클라우드 환경 |
| V-04 | 유사도 검색 API | 검색·조회 엔드포인트 |
| V-05 | 신규 특허 업데이트 파이프라인 | 주기적 증분 임베딩 |
| V-06 | 원격 모니터링·백업 | PatSol 담당 |

## 4. 시스템 구성

```
[애니파이브 IPBrain]
        │
        ▼  REST API (API Key 인증)
[PatSol API Gateway]
        │
        ├── 직무발명서 서비스 (PatSol 인프라)
        │       ├── F-01~03  파일 파싱 (이미지·PDF·PPT·DOCX)
        │       ├── F-04     LLM 초안 생성 (멀티모달)
        │       ├── F-05     편집기 렌더링
        │       └── F-06     다운로드 (DOCX·PDF·HWP)
        │
        └── VectorDB 서비스 (애니파이브 클라우드)
                ├── V-01~02  수집·임베딩 파이프라인
                ├── V-03     VectorDB
                ├── V-04     검색 API
                └── V-06     PatSol 원격 모니터링 ──→ PatSol
```

**인증 방식**: API Key + IP Whitelist (기본) / OAuth 2.0 (요청 시 추가)
