# 애니파이브 × Wand (PatSol) — 1차 기술 제안서

---

## 1. 제안 범위

| 항목             | 내용                                   |
| -------------- | ------------------------------------ |
| 직무발명서 생성 기능    | 멀티모달 입력 → AI 초안 생성 → 편집 → 다운로드       |
| 특허 VectorDB 구축 | 애니파이브 보유 특허 DB 전체 임베딩 + PatSol 원격 운영 |
| 연동 방식          | REST API                             |
| 인프라 소유 — 직무발명서    | Wand                                 |
| 인프라 소유 — VectorDB | 애니파이브 클라우드 계정 (내재화)                  |
| 운영 주체          | Wand 원격 운영·유지보수                      |

## 4. 시스템 구성

```
[애니파이브 IPBrain]
        │
        ▼  REST API (API Key 인증)
[PatSol API Gateway]
        │
        ├── 직무발명서 서비스 (Wand 인프라)
        │       ├── F-01~03  파일 파싱 (이미지·PDF·PPT·DOCX)
        │       ├── F-04     LLM 초안 생성 (멀티모달)
        │       ├── F-05     편집기 렌더링
        │       └── F-06     다운로드 (DOCX·PDF·HWP)
        │
        └── VectorDB 서비스 (애니파이브 클라우드)
                ├── V-01~02  수집·임베딩 파이프라인
                ├── V-03     VectorDB
                ├── V-04     검색 API
                └── V-06     PatSol 원격 모니터링 ──→ PatSol
```

**인증 방식**: API Key + IP Whitelist (기본) / OAuth 2.0 (요청 시 추가)

## 5. 개발 일정

### 단계 개요

| 단계 | 목표 | 예상 기간 |
|------|------|-----------|
| **PoC** | 핵심 기술 검증 — 텍스트 기반 직무발명서 생성 + 소규모 샘플 VectorDB 검색 동작 확인 | 착수 후 **4주** |
| **MVP** | 실사용 가능한 최소 기능 — 멀티파일 업로드·이미지 파싱·초안 생성·편집·DOCX/PDF 다운로드 + 실데이터 VectorDB 검색 API | PoC 이후 **+2~3개월** |
| **전체 완료** | HWP 출력·전체 IP 커버리지 임베딩·증분 업데이트 파이프라인·모니터링 체계·UAT 완료 | MVP 이후 **+2~3개월** |

### 단계별 범위

```
PoC (4주)
└── 직무발명서: 멀티파일 업로드·이미지 파싱·DOCX/PDF 다운로드·퀄리티 보장 x

MVP (PoC +2~3개월)
├── 직무발명서: 멀티파일 업로드·이미지 파싱·편집기·DOCX/PDF 다운로드·REST API
└── VectorDB: 실데이터 일부 임베딩·검색 API·애니파이브 클라우드 배포

전체 완료 (MVP +2~3개월)
├── 직무발명서: HWP 출력·임시저장·오류 처리 강화·퀄리티 보장
└── VectorDB: 전체 IP 커버리지 임베딩 완료·증분 업데이트 파이프라인·원격 모니터링 체계
```

> PoC 결과를 기반으로 MVP 범위 및 일정을 애니파이브와 공동 확정 하는 방향
