## 기능 개요 (What & Why)
검색 결과 페이지에서 AI가 추출한 토픽(Topic) 기반으로 특허 결과를 필터링하는 기능입니다.
사용자가 대량의 검색 결과 중 원하는 기술 주제에 해당하는 특허만 빠르게 추려볼 수 있도록
토픽 필터 섹션과 필터 칩(Filter Chip) UI를 제공합니다.

## 정책 및 규칙

### 필터 칩 상태
- 각 필터 칩은 Default / Selected 두 가지 상태를 가집니다.
- 클릭 시 Default ↔ Selected 토글됩니다.

### 다중 선택 시 논리 연산
- 선택된 칩이 1개: 해당 토픽만 포함된 특허 표시
- 선택된 칩이 2개 이상:
  - **OR 모드**: 선택된 토픽 중 하나라도 포함된 특허 표시 (기본값)
  - **AND 모드**: 선택된 토픽을 모두 포함한 특허만 표시
  - OR / AND 전환은 필터 섹션 상단의 토글 또는 버튼으로 제어합니다.
- OR / AND 기본값 및 UI 표현 방식은 미결 사항 참고.

### 검색 유형별 토픽 필터 노출 조건
- **키워드 검색**: 토픽 필터 섹션 표시
- **자연어 검색**: 토픽 필터 섹션 표시
- **조건식 검색**: 토픽 필터 섹션 **표시하지 않음**
  - 조건식 검색은 필드 지정 구조 검색이므로 AI 토픽 추출 결과와 충돌 가능성이 있어,
    현재 단계에서는 토픽 필터를 노출하지 않는 것으로 정의합니다.
  - 단, 조건식 검색 결과 화면에서 토픽 필터 영역이 빈 공간으로 보이거나
    에러처럼 보이지 않도록 레이아웃 처리가 필요합니다.

### 필터 초기화
- 검색어 변경 또는 신규 검색 실행 시 선택된 필터 칩 상태 초기화
